04. 이벤트 핸들러 제거

        EventTarget.prototype.removeEventListener 메서드를 통해 addEventL, hanistener 메서드로 
        등록한 이벤트 핸들러를 제거한다. 
        removerEventListener 메서드에 전달할 인수는 addEventListener 메서드와 동일하며
        전달한 인수가 일치하지 않을 경우 이벤트 핸들러는 제거되지 않는다
    
        이벤트 핸들러 프로퍼티 방식으로 등록한 이벤트 핸들러는 removeEventListener 메서드로
        제거할 수 없고 이벤트 핸들러 프로퍼티에 null을 할당햇서 이벤트를 제거한다
